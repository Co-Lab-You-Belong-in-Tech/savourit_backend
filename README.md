## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

### Install

- Ruby
- Ruby on Rails
- PostgresSQL

### Using it Locally

- Clone the project

```
git clone https://github.com/Co-Lab-You-Belong-in-Tech/savourit_backend
cd savourit_backend
```

### Setup

Install gems with:

```
bundle install
```

Setup database with:

```
rails db:create
rails db:migrate
rails db:seed
```

### Usage

Start server with:

```
rails server -p 3001
```

Visit http://localhost:3001/ in your browser.
